Pseudo clases pseudo elementos y atributos

Pseudo clases (:)

Una pseudo clase selecciona elementos en un estado o posicion, es decir cuando el mouse se coloca sobre ellos. comunmente se utiliza para estilizar enlaces

tipos mas comunes

  • :hover cuando el mouse pasa por encima del elemento
  • :focus cuando un elemento esta enfocado (por tecla o click)
  • :first-child si es el primer hijo de su padre
  • :last-child si es el ultimo hijo de su padre
  • :nth-child(n) si es el hijo numero n de su padre

Pseudo elementos (::)

Un pseudo elemento permite utilizar partes especificas de un elemento

tipos mas comunes

  • ::before Inserta contenido antes del elemento
  • ::after Inserta contenido despues del elemento
  • ::first-letter Selecciona la primera letra del texto
  • ::first-line Selecciona la primera linea del texto
  • ::placeholder Estiliza el tecto de un placeholder en un input

Selectores de atributos ([])

Permite aplicar estilos a los elementos que tienen atributos especificos o valores determinados
