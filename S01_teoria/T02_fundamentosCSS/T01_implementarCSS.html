Formas de implementar CSS

Existen 3 formas de aplicar estilos a HTML

1. En linea (Inline css)

Se escribe directamente en la etiqueta HTML, usando el atributo style

Ejemplo:

Hola mundo con etiqueta style en la misma linea

2. Interno o enbebido (Internal css)

Se coloca dentro de una etiqueta style en el head del documento HTML

Ejemplo:

Hola mundo con etiqueta style enbebido en el head

3. Externo (External css)

Se define en un archivo aparte por Ejemplo, estilos.css y se enlaza con el HTML mediante un etiqueta link en el head

Ejemplo:

Hola mundo con un archivo externo con link en el head
