Etiquetas div, section, article y span

Etiqueta div(división genérica)

Es un contenedor genérico que no tiene un significado semántico propio, pero sirve para organizar y estructurar el contenido

Etiqueta section (sección de contenido)

Etiqueta de bloque, sirve para agrupar contenido que forma parte de una misma temática. dividir un articulo en secciones como "Introducción", "contenido", "conclusion"

Etiqueta article (contenido independiente)

Etiqueta de bloque, que tiene sentido por si solo. por ejemplo: noticias, publicaciones, comentarios

span (contenedor en linea)

Etiqueta en linea, sirve para aplicar estilos en linea a palabras o frases dentro de un parrofo, estoy dentro de un span con estilo color rojo

Atributo class (clase)

Definición

Se usa para asignar uno o mas nombres de clase a un elemento, estos nombres se pueden usar en CSS o JavaScript

Caracteristicas

  • Se pueden repetir en muchos elementos
  • un mismo elemento puede tener varias clases separadas por espacios
