Atajos Emmet para html

HTML

Con emmet puedo crear rapidamente una plantilla HTML. En un archivo HTML, escribir ! seguido de TAB

Etiquetas basicas

Para crear etiquetas HTML basicas, escribimos el nombre de la etiqueta y presionamos ENTER

  • div: contedor
  • p: parrafo
  • ul: lista desordenada
  • li: lista ordenada
  • a: enlace
  • bq: para blockquote
  • hdr: para header
  • ftr: para footer
  • btn: para botton
  • sect: para una seccion

clases y id

Para referirse a las clases emmet usa el punto .

  • div.wrapper div con una clase wrapper

Para referirse a los id emmet usa el numeral #

  • div#hero div con un id hero

Hermanos

Para referirse a etiquetas hermanos +

  • h1+h2+h3 etiquetas al mismo nivel

anidados

Para referirse a etiquetas hijos o aninados >

  • ul>li etiquetas li es hijo de ul
  • na>ul>li*3

escalonado

querermos que la etiqueta blockquote estuviera al mismo nivel que el parrafo. por eso vamos a trepar con ^ (alt + 94)

  • div+div>p>span+em^bq

agrupar

crearemos un encabezado y un pie de pagina usando parentesis ( )

  • div>(header>ul>li>a)+footer>p

multiplicacion y $

Podemos usar multiples etiquetas multiplicando (*) y numerando elementos en secuencia usando el signo del dolar ($)

sin numeracion: ul>li*5

con numeracion: ul>li.item$$$*5

comience con un numero especifico: ul>li.item$@3*5

direccion inversa: ul>li.item$@-*5

direccion inversa desde un numero especifico: ul>li.item$@-3*5
