FORMULARIO

Los campos de un formulario van dentro de una etiqueta form porque esta etiqueta define el contexto del formulario. Le dice al navegador que todos esos campos estan relacionados y que deben enviarse juntos


Porque usamos la etiqueta form

  1. Agrupar los campos

    2. Todos los que estan dentro de la etiqueta se considera parte del mismo formulario

  2. Permite el envio de datos

    3. Al hacer click en el boton de tipo submit, el navegador recopila los datos de todos los campos dentro de form y los envia a una URL (servidor) que se define en el atributo action

  3. Manejo de metodos HTTP

    4. Puedes especificar como enviar los datos con el atributo method:

    • GET: envia los datos por URL(se ve en la barra de navegacion).
    • POST: envia los datos de forma oculta (seguro para contraseñas)
  4. Integracion con validaciones

    5. El navegador aplica validaciones automaticas antes de enviar un formulario
