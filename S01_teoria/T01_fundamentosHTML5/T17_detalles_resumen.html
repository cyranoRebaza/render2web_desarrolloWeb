Detalles y resumen

Etiqueta details (detalles)

La etiqueta details define un elemento interactivo que el usuario puede abrir y cerrar

  • Al estar cerrado solo se muestra el encabezado(si se define con summary )
  • Al abrirla, se muestra todo el contenido que contiene

Etiqueta summary

La etiqueta summary define el titulo visible del bloque details

  • Debe ir dentro de details

Ejemplo:

¿Qué es HTML?

HTML es un lenguaje de marcado que se usa para estructurar contenido en la web.

¿Qué es CSS?

CSS es un lenguaje de estilos que se usa para diseñar y dar formato a las páginas web.
