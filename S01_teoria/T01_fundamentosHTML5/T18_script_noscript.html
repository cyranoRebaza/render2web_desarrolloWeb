Etiqueta script y noscript

Etiqueta script

La etiqueta script sirve para inscrustar codigo JavaScript en un documento HTML o para enlazar un archivo externo de JavaScript

  • para que se usa:
    • Agregar interactividad a la pagina
    • Ejecutar codigo al cargarse el documento
    • Manipular el DOM, realizar validaciones
  • formar de usarlo:
    • Codigo JavaScript embebido en el HTML
    • Enlazar un archivo JavaScript externo

Etiqueta noscript

La etiqueta noscript sirve para mostrar contenido alternativo a los usuarios

  • para que se usa:
    • cuando tienen desactivado JavaScript en su navegador
    • usan un navegador que no soporta js
  • para que sirve:
    • Informar al usuario que necesita habilitar JS
    • Ofrecer contenido basico en caso de que JS este desabilitado
